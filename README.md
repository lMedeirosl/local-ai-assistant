# Local AI Knowledge Assistant

A local AI assistant that uses a Large Language Model (LLM) to answer questions based on custom documents.

## About the project

This project is a simple AI assistant developed in Python.

The user can add documents to the knowledge base, ask questions through the terminal, and receive answers generated by a local LLM.

The project uses Ollama to run an AI model locally without requiring external APIs.

## Features

- Read custom text documents
- Use local documents as knowledge
- Ask questions through the terminal
- Generate answers using a local LLM
- Fully offline AI inference

## Technologies

- Python
- Ollama
- Qwen2.5 Coder 3B
- Linux
- Virtual Environment (venv)

## How it works

1. The application loads documents from the `documents` folder.

2. The user asks a question through the terminal.

3. The question and document context are sent to the local LLM.

4. The AI generates an answer using the provided information.

## Installation

Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
```
Enter the project folder:
```
cd local-ai-assistant
```
Create a virtual environment:
```
python -m venv venv
```
Activate the environment:
```
source venv/bin/activate
```
Install dependencies:
```
pip install -r requirements.txt
```
Start Ollama and make sure the model is available:
```
ollama list
```
Run the assistant:
```
python src/main.py
```
